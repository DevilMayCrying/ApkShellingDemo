# ApkShellingDemo
使用 xposed 框架 ，获取通过 360加固、梆梆加固、腾讯乐固、百度加固 这种免费加固过应用的 dex文件。实现反编译


## 注意
1. 在调试三方apk的时候要其 sdcard 存储权限因为 dex文件会存放在 sdcard中

2. 项目重新运行后 需要重启手机 才会生效，这是因为 xposed框架的原因
